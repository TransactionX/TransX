

计算btc的本次金额算力：
	1. btc平均算力 av = 昨天所有币种算力总和/矿机数

	2. 本次的初步算力 first_pow = 膨胀金额 * 挖矿类型占比

	3. 判断今天的挖矿是否超过平均算力 （if 该矿机今天所有币种累计算力 + first_pow >= av）
		不超过：本次算力 = first_pow
		超过：进行下一步


	4.

	n = （该矿机今天的所有币种算力累计 + first_pow）/ av  # 向上取整

	钝化系数 = n * 1.5^n

	本次算力 = 金额奖励占比 * first_pow / 钝化系数

	本次算力 > 昨天总算力？？？？？？？？ 对的

	注意：这里说的"昨天"指的是对应总算力不是0的那天
